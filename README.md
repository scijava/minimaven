# MiniMaven

MiniMaven is a minimal build system interpreting Maven-style pom.xml files.
Its primary purpose is to support the SciJava scripting framework
("Java as a scripting language").

Its initial purpose was to serve as a drop-in replacement for Maven to build
Fiji, ImageJ2 and other very simple Java projects. MiniMaven was originally
maintained as part of [Fiji](https://fiji.sc/) as a successor to the Fiji Build
System, it migrated to [ImageJ2](https://imagej.net/software/imagej2) and
subsequently to the [SciJava](https://scijava.org/) family of projects.
